# 1. 프로젝트 개요

마도 거래소는 주식을 모티브로한 판타지 세계의 주식 게임입니다. 해당 게임으로서 유저들은 즐겁게 주식에 대한 공부를 할 수 있으며, 기본적인 주식에 대한 흐름을 알 수 있게 됩니다.

+ 가상의 주식 게임을 구상하면서 사용자들에게 주식에 대한 지식을 제공
+ 주식 게임의 세계를 판타지 세계로 구상하여 누구나 재미있는 접근을 허용
+ 뉴스 이벤트를 통해서 실시간 변동 이벤트를 제공
+ 마치 실제로 움직이는 듯한 변동으로 사용자들에게 몰입감을 제공
